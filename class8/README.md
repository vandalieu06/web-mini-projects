# Desenvolupament web en entorn client

- Realitzar una web per guardar les notes dels mòduls i disposar d’una mitja. Requisits:
- Funció que demanarà el mòdul i la seva nota 
- Validar camps. 
- Es mostraran les dades a la web en temps real. 
- Es mostrarà la mitjana en temps real i s’anirà actualitzant. 
- Les dades sempre han d’estar disponible a la web i seran acumulatives (localStorage). 
- Botó “X” esborrarà el mòdul i la nota corresponent. 
- Tot documentat.
